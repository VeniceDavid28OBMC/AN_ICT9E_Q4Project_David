WHAT DO I EXPECT FOR ICT 10?

1. WEB DEVELOPMENT

Web development is the backbone of building modern websites and applications. Entering Grade 10 is a big step, and I believe we will be able to foster web development. I will gain better skills which I can use in my codings.

2. MULTIMEDIA AND GRAPHICS

I expect to dive into making beautiful graphics in coding. With each project, we grow more confident in our creative abilities. Through CSS, we will be able to showcase our artistic side.

3. COMPLEX JAVASCRIPT

I think complex JavaScript will be tough but rewarding. I expect to learn how to add cool interactive features to websites using functions. JavaScript is essential for turning lifeless pages into interactive digital experiences.

4.FUN CODING

I believe coding this year will be fun and exciting, not just boring lessons. I expect to work on games or interactive projects that make learning enjoyable.
